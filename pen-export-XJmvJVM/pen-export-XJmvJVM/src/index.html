시작하기 ✅ 체크리스트

아래 항목을 하나씩 완료해 보세요. 체크 내역은 이 브라우저에 저장됩니다.

① 구글 첫 로그인

1) 계정 정보 확인

  1. 학교에서 받은 구글 계정과 임시 비밀번호를 준비하세요.
  2. 컴퓨터/태블릿에서 Chrome을 열고 사생활 모드(Ctrl+Shift+N) 권장.
💡 다른 계정이 로그인되어 있다면, 우측 상단 프로필에서 로그아웃 후 진행하세요.

2) 로그인 & 비밀번호 변경

  1. accounts.google.com 접속
  2. 학교 G메일 주소 입력 → 임시 비밀번호 입력
  3. 요청 시 새 비밀번호로 변경 (8자 이상, 추천: 영문 대·소문자+숫자)
  4. 2단계 인증이 설정된 학교라면 가이드에 따라 진행

3) 프로필/언어/시간대 설정

  1. 우측 상단 프로필 → Google 계정 관리
  2. 개인정보의 언어/시간대를 한국어/서울(UTC+9)로 확인
  3. 프로필 사진 등록(선택)

② 클래스룸 입장 & 과제

1) 수업 참여

  1. classroom.google.com 접속
  2. ＋ 참여 → 수업 코드 입력
  3. 수업 카드가 보이면 입장

2) 과제 확인 & 제출

  1. 스트림에서 공지 확인, 과제 탭에서 과제 목록 확인
  2. 과제 열기 → 첨부파일 추가(드라이브, 링크, 파일 업로드 등)
  3. 제출 클릭 (필요 시 제출 취소 후 파일 수정 가능)
파일 형식 팁
  • 문서는 Google Docs, 표는 Sheets, 발표는 Slides 권장
  • 사진/동영상 업로드 시 용량이 크면 드라이브 링크 공유

3) 알림 & 댓글 예절

  • 과제 마감 알림: 클래스룸 우측 상단 종 모양 아이콘
  • 개별 댓글은 교사-학생 1:1 소통, 공개 댓글은 모두가 보니 예의있게

③ 드라이브 기본

1) 폴더 정리 규칙

  • 학년/과목/날짜 기준으로 폴더 생성 (예: 1학년/과학/2025-09/)
  • 공유가 필요한 폴더는 공유 설정에서 권한(보기/편집) 확인

2) 업로드 & 공유

  1. 드라이브 열기 → 새로 만들기 → 폴더/파일 업로드
  2. 파일 우클릭 → 링크 공유 → 권한 설정(제한/링크가 있는 모든 사용자)
  3. 클래스룸 과제에 링크 첨부하기

④ 문서(Docs) 작성

1) 새 문서 만들기

  1. 드라이브 → 새로 만들기 → Google 문서
  2. 제목 클릭해 문서 이름 바꾸기 (예: 과학_탐구보고서_홍길동)

2) 서식 빠르게 맞추기

  • 스타일에서 제목/부제/본문 통일
  • 삽입 → 표/이미지/페이지번호 활용
  • 공유 → 보기/댓글/편집 권한 설정
제출용 표지 예시 
제목: ○○보고서
이름: ○○○(학번 ○번)
과목: 과학
제출일: 2025-09-15

⑤ 시트(Sheets) 작성

1) 새 시트 만들기

  1. 드라이브 → 새로 만들기 → Google 스프레드시트
  2. 제목을 과학_자료정리_홍길동 처럼 구체적으로

2) 자주 쓰는 기능

  • 열 너비 맞춤: 열 머리글 더블클릭
  • 첫 행 고정: 보기 → 고정 → 1개 행
  • 정렬: 데이터 → 정렬 조건
기본 수식 예시 
=SUM(B2:B10)         합계
=AVERAGE(C2:C20)     평균
=COUNTIF(D2:D100,"제출")  '제출' 개수
=IF(E2>=60,"합격","재제출")

3) 공유 & 제출

  1. 오른쪽 상단 공유 → 사용자 추가 또는 링크 공유
  2. 클래스룸 과제에 링크 첨부 → 제출

⑥ 키보드 단축키 (유용!)

공통

  • 복사: Ctrl+C / 붙여넣기: Ctrl+V
  • 실행 취소: Ctrl+Z / 다시 실행: Ctrl+Y
  • 검색: Ctrl+F

Docs

  • 굵게: Ctrl+B, 기울임: Ctrl+I
  • 머리말 적용: Ctrl+Alt+1~6

Sheets

  • 현재 날짜: Ctrl+;
  • 행/열 추가: Ctrl+Alt+=

자주 묻는 질문 (FAQ)

클래스룸 수업 코드가 안 된대요.

코드를 정확히 입력했는지 확인하고, 학교 계정으로 로그인했는지 점검하세요. 여전히 안 되면 선생님께 코드/권한을 확인 요청하세요.

파일을 올리면 용량이 너무 커요.

동영상은 드라이브에 올리고 링크를 첨부하세요. 이미지도 압축해서 올리면 좋아요.

문서/시트가 편집이 안 돼요.

공유 권한이 보기로 되어 있을 수 있어요. 소유자에게 편집 권한을 요청하세요.
